# RNAtools
RNAseq scripts.

Type `--help` to see the usage of scripts, e.g.
```R
Rscript Volcano.R --help
```

**Requirements**  
R >= 4.0  
* AnnotationDbi  
* argparse  
* biomaRt  
* ComplexHeatmap  
* DESeq2  
* GenomicFeatures  
* ggrepel  
* KEGGREST  
* org.Hs.eg.db  
* this.path   
* tidyverse  
* tximport  
* viridis
