# RNAtools
RNAseq scripts.

Type `--help` to see the usage of scripts, e.g.
```
Rscript Volcano.R --help
```

**Requirements**  
R >= 4.0  
R packages:  
AnnotationDbi  
argparse  
biomaRt  
ComplexHeatmap  
DESeq2  
GenomicFeatures  
ggrepel  
KEGGREST  
org.Hs.eg.db  
tidyverse  
tximport  
viridis
